AddExtension (str, ext)
Adds an extension to a string, as _str.ext_, e.g. for filenames.

If str already has the dot and extension, this is a no-op.

Parameters:

  • str string String to extend.
  • ext string Extension. The first character may be a dot, but this is not necessary.

Returns:

    string Extended string.
BeginsWith (str, patt, get_suffix)
Predicate.

Parameters:

  • str string Source string.
  • patt string Prefix pattern. (n.b. only supports exact "patterns")
  • get_suffix bool Get the rest of the string as well?

Returns:

  1. boolean str begins with patt?
  2. string If patt was found and get_suffix is true, the rest of the string; otherwise, the empty string.
